创建远程分支
1、从本地推送到远程
如果你有一个名为feature-x的本地分支，并想将其推送到远程仓库
git push origin 分支名
git push origin feature-x

2、直接在远程创建新分支
git push origin HEAD:new-branch-name

3、查看远程分支
git branch -r

4、切换到远程分支
要切换到远程分支的本地副本（如果本地还没有这个分支），你可以使用：
git checkout -b 本地分支名 origin/远程分支名

在某个分支下创建新分分支，新分支初始化基于当前分支
当前分支branch3
git checkout -b branch3-part1
 
git push --set-upstream origin branch3-part1 推送到远程仓库
